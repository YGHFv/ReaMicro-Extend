# 阅微补全计划（ReaMicro Extend）

阅微补全计划（ReaMicro Extend）是一个面向阅微的 LSPosed 扩展模块，用来补全阅微在搜索、在线源、云盘、阅读、文件编辑和字体管理上的一些日常使用能力。

## 功能概览

### 在线补全

- 在阅微设置中增加独立的「在线补全」页面，在线源与关联补全源分开管理。
- 支持导入阅读/Legado 书源格式的在线源：
  - 点击「添加在线源」默认读取剪贴板链接导入。
  - 长按「添加在线源」可选择单个源文件导入。
  - 一次只允许导入一个源，检测到多个源会取消导入。
- 每个在线源可单独启用/停用。需要账号密码的源会弹出登录框；暂不支持登录的源允许跳过后启用。
- 阅微首页搜索会并发搜索已启用的在线源，搜索分组显示为「在线补全-源名」。
- 搜索结果按阅微原有搜索分组方式展示：每个源默认显示首个/最佳结果，可通过列表展开按钮展开更多结果。
- 在线搜索结果展示封面、书名、作者，以及连载状态、字数、章节数、更新时间等可获取信息。
- 点击在线搜索结果会后台下载并导入阅微：
  - 小于 200 章的图书下载完成后一次性导入。
  - 大于 200 章的图书先下载前 100 章并导入书架，后续章节继续后台下载并写入。
  - 下载期间可继续阅读其他图书或切到其他应用。
  - 通知栏显示章节进度，并提供取消按钮；取消或异常中断后会清理临时下载缓存。
  - 下载失败的章节会自动重试，仍失败时跳过继续下载，后续再重试。
- 在线导入会生成标准 EPUB 结构，逐章写入 `xhtml`，生成目录与元数据，并清理正文开头重复章节标题。
- 支持在线源封面下载与显示，包括同源 `http` 封面自动升级为 `https`、必要时放行明文封面请求。
- 在线补全导入的图书支持在图书菜单中执行「更新章节」，从对应在线源获取目录并追加新增章节。

### 关联补全

- 在阅微原有关联搜索基础上补充更多关联搜索源，并按「关联补全-源名」显示结果。
- 支持按源启用/停用关联补全搜索源。
- 恢复图书详情页关联操作入口，支持取消关联、重新关联和封面修复。
- 支持手动编辑关联数据入口（需要对应功能源存在）。

### WebDAV 与本地书库

- 在阅微云盘体系中补充 WebDAV 入口，支持 WebDAV 登录、浏览、导入、下载与备份。
- 优化云盘/本地书库列表展示，补充图标、标题和部分状态显示。
- 支持本地书库索引与搜索刷新。
- 支持云盘下载取消，取消后同步清理进行中的任务状态。
- 增加导出到本地存储能力，方便把阅微中的图书导出到本地目录。

### 阅读页增强

- 阅读页自动翻页。
- 导入覆盖检查，避免重复导入时误覆盖已有图书；在线补全的阶段性导入走静默更新。
- 阅读页全文搜索增强：
  - 建立本地索引。
  - 搜索结果列表可定位当前项。
  - 跳转后注入高亮，并在高亮未渲染到可见页面时自动纠正翻页。
- 对 ScrollPager 崩溃增加兜底：检测到上次滚动翻页崩溃后自动回退到平移翻页。
- 补充部分阅读 DOM 渲染兼容处理，降低特定书籍排版异常导致的崩溃概率。

### EPUB 文件编辑

- 增强阅微文件编辑能力，支持在图书内编辑 EPUB 元数据和文件内容。
- 支持查看和编辑 `xhtml`、`html`、`xml`、`opf`、`ncx`、`css` 等文本文件。
- 支持文件内搜索、结果跳转、替换和未保存修改确认。
- 支持预览图片和字体文件。
- 支持修改封面。
- 支持在图片菜单中将图片设置为阅微横图，自动按阅微横图命名规则重命名并刷新缓存。
- 支持新增、重命名、删除 EPUB 内部文件，并在文件变化后刷新缓存。

### 字体补全

- 支持字体库管理，可导入 `ttf` / `otf` 字体文件。
- 支持设置全局界面字体。
- 支持宋体、楷体映射，改善阅读排版中缺字或字体不一致的问题。
- 字体库与阅读排版设置互通。

### 账号与数据

- 支持导出当前账号登录凭证。
- 支持从剪贴板或文件导入凭证。
- 支持保存并切换多个账号。
- 支持导入/导出账号数据包。
- 支持启动时清理部分缓存状态，降低异常中断后的残留影响。

### 屏幕方向

- 在阅微设置中增加旋转补全：
  - 自动旋转
  - 竖向锁定
  - 横向锁定
  - 反向旋转

## 使用方式

1. 安装模块 APK。
2. 在 LSPosed 中启用模块，并勾选阅微作用域。
3. 强制停止并重新打开阅微。
4. 进入阅微设置，打开「阅微补全计划」相关开关。
5. 如需在线源，进入「在线补全」页面导入并启用在线源。

通知相关功能需要授予模块通知权限。在线补全下载可以在后台运行，部分系统可能还需要允许模块后台启动或关闭过度省电限制。

## 构建

需要：

- JDK 17
- Android SDK

运行单元测试：

```bash
./gradlew :app:testDebugUnitTest
```

构建调试版：

```bash
./gradlew :app:assembleDebug
```

构建正式版：

```bash
./gradlew :app:assembleRelease
```

如果需要本地签名正式版，可通过环境变量或 `signing/reamicro-release-secrets.txt` 配置：

```properties
RELEASE_KEYSTORE_FILE=signing/reamicro-release.jks
RELEASE_KEYSTORE_PASSWORD=...
RELEASE_KEY_ALIAS=...
RELEASE_KEY_PASSWORD=...
```

`local.properties` 需要指向本机 Android SDK，例如：

```properties
sdk.dir=C:/Users/<name>/AppData/Local/Android/Sdk
```
